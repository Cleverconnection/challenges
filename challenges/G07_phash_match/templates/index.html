Correspondência perceptual

Este painel exibe os hashes perceptuais calculados para imagens históricas do ImageNet.

Envie um JSON para /comparar com o pHash que deseja testar. Se corresponder à foto da inauguração do ImageNet, a flag será revelada.

Exemplo de requisição

curl -X POST http://host:5007/comparar \
  -H "Content-Type: application/json" \
  -d '{"phash":"abcdef1234567890"}'

A resposta incluirá o nome da imagem com a menor distância de Hamming.
