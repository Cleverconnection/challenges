Weaving the Tree

Radia Perlman apelidou o Spanning Tree Protocol de “poesia em movimento”. Nesta simulação, você recebe os sinais brutos de uma malha que está retomando o equilíbrio após uma eleição de root bridge disputada por duas bridges de laboratório.

Baixe o script de geração ou replique os dados manualmente. A flag segue o formato CECYBER{root_<bridgeID>_<portID>} e pode ser validada no endpoint /verify.

  • GET /pcap-script — retorna o gerador em Python que produz o PCAP.
  • GET /bpdu — lista os campos relevantes dos BPDUs que inspiraram a captura.
  • POST /verify — envia sua flag em JSON ({"flag": "..."}).
