# Backup Público

## Visão geral
**Categoria:** Web

## Tabela de referência interna

| Desafio | Flag | Dificuldade |
| ------- | ----- | --------- |
| Backup Público | `CTF{public_backup_flag}` | Fácil |

## Execução
1. Um arquivo de backup foi deixado acessível na raiz do serviço web. Ele contém artefatos sensíveis do ambiente, incluindo informações que levam à flag.
2. URL de acesso individual fornecida ao iniciar a instância do desafio.
3. Baixar o pacote, extrair e recuperar a flag indicada.
4. `CTF{public_backup_flag}`
5. Baixe `backup.zip`, extraia e procure por flag/proof no conteúdo.

## Narrativa
Explorar Backup Público revela a flag quando a vulnerabilidade principal é compreendida e explorada.
