Atlas de Visão

Fei-Fei Li liderou a criação do ImageNet, um atlas gigantesco de imagens anotadas que redefiniu a visão computacional. O laboratório pede ajuda para reconstruir a etiqueta principal usada na primeira versão pública do conjunto.

Ferramentas liberadas:

  • GET /dataset — retorna segmentos com palavras-chave que, combinadas, formam a etiqueta principal.
  • GET /annotations — mostra tags auxiliares usadas no laboratório.
  • POST /vision — envie um JSON com a lista {"tags": [...]}. Inclua a etiqueta montada para receber a flag.

Monte a tag histórica seguindo as pistas e lembre-se de usar letras maiúsculas. Ao enviar um JSON contendo a string correta, o serviço responde com a flag.

Formato da flag: CTF{...}.
