# Envio Público

## Visão geral
**Categoria:** Web

## Tabela de referência interna

| Desafio | Flag | Dificuldade |
| ------- | ----- | --------- |
| Envio Público | `CTF{predictable_upload_leak}` | Fácil |

## Execução
1. Uma central de uploads publica arquivos enviados em uma pasta estática com nomes previsíveis. Documentos internos foram deixados ao lado da área pública.
2. URL de acesso individual fornecida ao iniciar a instância do desafio.
3. Determinar o caminho de um arquivo sensível e utilizá-lo para obter a flag.
4. `CTF{predictable_upload_leak}`
5. Liste ou acesse nomes previsíveis e procure por arquivos contendo "flag".

## Narrativa
Explorar Envio Público revela a flag quando a vulnerabilidade principal é compreendida e explorada.
