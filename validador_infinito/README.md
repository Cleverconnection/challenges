# Validador Infinito

## Visão geral
**Categoria:** Web

## Tabela de referência interna

| Desafio | Flag | Dificuldade |
| ------- | ----- | --------- |
| Validador Infinito | `CTF{rate_limit_none}` | Fácil |

## Execução
1. Um verificador de código temporário permite tentativas ilimitadas e usa um espaço de busca reduzido. Sem bloqueio progressivo, é viável adivinhar o código por força bruta controlada.
2. URL de acesso individual fornecida ao iniciar a instância do desafio.
3. Realizar tentativas até encontrar o código aceito e obter a flag.
4. `CTF{rate_limit_none}`
5. Use paralelismo moderado e monitore a primeira resposta de sucesso.

## Narrativa
Explorar Validador Infinito revela a flag quando a vulnerabilidade principal é compreendida e explorada.
